# Servo-Control-GUI-v1.0.0
# Servo Control GUI (ST3215) - (Feetech/Waveshare)

A simple Windows GUI to control ST3215 servo motors via USB-to-Serial.

---

## Download

Go to **Releases** and download the latest version.

---

## Features

- COM port selection
- Servo scanning
- Position control (slider / raw / degrees)
- Home position
- Change servo ID

---

## How to Use

1. Download the ZIP from Releases  
2. Extract it  
3. Run the EXE  

---

## Requirements

- Windows OS  
- USB-to-Serial driver (CH340 / FTDI)
